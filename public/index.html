HOW TO USE
Fix mode — Paste a GitHub repo URL or select from dropdown, Scan, Apply fixes, Create PR
Feature mode — Select repo, Load files, describe feature in plain English, Generate, Create PR
Agent mode — Select repo, set duration, add tasks, Activate agent, go for lunch
God mode — Scan live URL, scan deps, sweep multiple repos, voice commands, auto-merge
Rules tab — Write custom lint rules, publish to marketplace, install community packs
Schedules tab — Set cron expressions for automatic nightly or weekly scans
Tests tab — Select files, generate Jest/pytest/JUnit tests, commit as PR
Voice — Say "scan my repo", "fix all issues", "create PR", "god mode"
