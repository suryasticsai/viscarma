📖 Technical Documentation

VisCarma is a multi‑agent system that autonomously improves your codebase. It combines the power of large language models (via Ollama) with battle‑tested tools like Aider, ESLint, and Playwright to fix bugs, add features, and orchestrate GitHub pull requests – all offline and private.

📦 Quick Install

git clone https://github.com/suryasticsai/viscarma.git
cd viscarma
npm install
python -m pip install aider-chat
node server.js

Then open http://localhost:9000

🚀 Usage

  1. Enter repo, owner, branch, and a mission prompt.
  2. Click Launch Mission – watch the agents collaborate in real‑time.
  3. A pull request will be opened on your repo.
