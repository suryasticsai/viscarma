HOW TO USE
Fix mode — Paste a GitHub repo URL or select from dropdown → Scan → Apply fixes → Create PR
Feature mode — Select repo → Load files → Describe feature in plain English → Generate → Create PR
Agent mode — Select repo → Set duration (e.g. 45 min) → Add tasks → Activate agent → Go for lunch ☕
God mode — Scan any live URL · Scan dependencies · Sweep multiple repos · Voice commands · Auto-merge
Rules tab — Write custom lint rules, publish to marketplace, install community packs
Schedules tab — Set cron expressions for automatic nightly or weekly scans
Tests tab — Select files, generate Jest/pytest/JUnit unit tests, commit as PR
Voice — Say "scan my repo", "fix all issues", "create PR", "god mode" and agent executes
TIPS
Add [![VisCarMa](https://viscarma.onrender.com/badge/owner/repo.svg)](https://viscarma.onrender.com) to any README
Set VISCARMA_PRO_USERS=yourusername in Render env for unlimited scans
Click Explain in the header to get an AI walkthrough of the current file
Click Install app (when visible) to add VisCarMa to your home screen
In Agent mode, Jira tickets are auto-converted to tasks — just enter your project key
