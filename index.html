🕵️ About VisCarma

VisCarma is an autonomous AI agent team that fixes bugs, invents features, and orchestrates code changes on your GitHub repositories. It uses a multi-agent system powered by Ollama, Aider, and Playwright – all running locally on your machine.

🤖 The Agents

  • VisCarma – Supreme Commander: oversees missions and provides final approval.
  • Parsh – Bug Killer: identifies and eliminates code bugs and errors.
  • Krish – Feature Inventor: designs and implements new capabilities.
  • Parth – The Driver: routes tasks to the right agent and orchestrates the workflow.

📖 How to Use

  1. Enter the GitHub repo name, owner, and base branch (defaults are pre-filled).
  2. Write a mission prompt (e.g., "Fix all ESLint issues", "Add a to-do list").
  3. Click Launch Mission – watch the agents collaborate in real-time.
  4. A pull request will be opened on your repo with the changes.
